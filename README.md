# Bukkit Host

Ultra-Low RAM Minecraft Bedrock Edition server optimized for performance and simplicity.

## Features

- Minecraft Bedrock Edition server software
- Optimized for minimal RAM usage (256MB-512MB)
- Full vanilla gameplay with mobs, commands, and console
- Support for multiple worlds (Overworld, Nether, End)
- Low-latency network configuration
- Easy to use and deploy

## Quick Start

```bash
bash start.sh
```
